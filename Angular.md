# Angular

## PIPES

### How use?

Los pipes solo transforman las variables de manera visual, no alteran el valor de la variable.

Pipes only transform the visual shape of variables, they don't change the value of the variabe.

~~~html
<h1>
    {{ nameVar | namePipe }}
</h1>
~~~

#### GENERIC PIPES

* upperCase
* lowerCase
* titlecase
* number
* json
